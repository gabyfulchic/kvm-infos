# KVM

### LAUNCH THE PROJECT  
### It will create 2 Centos7 Vm by default in a Bridged Network  

```ruby
git clone https://github.com/gabyfulchic/kvm-infos.git
cd kvm-infos/
chmod +x deploy-vm.sh \
    destroy-vm.sh \
    new-bridge.sh
./deploy-vm.sh
```
